# 微信小程序导航栏和分享功能说明

## 📱 原生导航栏（当前配置）

### 什么是原生导航栏？
使用 `navigationStyle: "default"` 时，微信小程序会自动提供：
- ✅ **左侧**：页面标题（可配置）
- ✅ **右侧**：胶囊按钮（微信自动提供，包含关闭、转发等功能）

### 胶囊按钮包含的功能：
1. **关闭按钮** - 点击退出小程序
2. **更多菜单**（三个点）- 包含：
   - 转发（分享给好友）
   - 添加到我的小程序
   - 设置
   - 反馈与投诉
   - 关于小程序

### ⚠️ 重要提示
**胶囊按钮只在以下环境显示：**
- ✅ 真机预览/真机调试
- ✅ 微信开发者工具的"真机调试"模式
- ❌ H5 预览模式（不会显示）
- ❌ 浏览器预览（不会显示）

## 🔄 分享功能实现

### 现代实现方式（已配置）

在页面的 script 中添加以下生命周期函数：

```javascript
// 分享给好友
onShareAppMessage() {
    return {
        title: '景区预约 - 轻松预约，畅游无忧',
        path: '/pages/index/index',
        imageUrl: '' // 可选：自定义分享图片
    }
}

// 分享到朋友圈（可选）
onShareTimeline() {
    return {
        title: '景区预约 - 轻松预约，畅游无忧',
        query: '',
        imageUrl: ''
    }
}
```

### 分享功能说明

1. **分享给好友**
   - 用户点击右上角胶囊按钮 → 转发
   - 自动调用 `onShareAppMessage()` 方法
   - 返回分享的标题、路径和图片

2. **分享到朋友圈**
   - 需要在页面中定义 `onShareTimeline()` 方法
   - 用户点击右上角胶囊按钮 → 分享到朋友圈
   - 只有定义了此方法，才会显示"分享到朋友圈"选项

3. **主动触发分享**（可选）
   ```javascript
   // 在页面中添加分享按钮
   uni.showShareMenu({
       withShareTicket: true,
       menus: ['shareAppMessage', 'shareTimeline']
   })
   ```

## 🚫 早期方式（不推荐）

早期小程序开发中，开发者会：
1. 设置 `navigationStyle: "custom"` 隐藏原生导航栏
2. 自己实现导航栏、返回按钮、分享按钮
3. 手动处理状态栏高度、安全区域等

**现在不需要这样做了！** 微信小程序的原生导航栏已经非常完善，直接使用即可。

## 📝 测试步骤

1. **在微信开发者工具中运行**
   - 确保选择的是"微信小程序"模式
   - 右上角应该能看到胶囊按钮（关闭 + 三个点）

2. **测试分享功能**
   - 点击右上角的三个点
   - 选择"转发"
   - 应该能看到自定义的分享标题和内容

3. **真机测试**
   - 点击"预览"生成二维码
   - 用微信扫码打开
   - 查看真机上的效果

## 💡 自定义导航栏（高级用法）

如果你需要完全自定义导航栏样式，可以：

```json
{
    "style": {
        "navigationStyle": "custom"  // 自定义导航栏
    }
}
```

但需要注意：
- 需要自己处理状态栏高度
- 需要预留胶囊按钮的位置（胶囊按钮仍会显示）
- 需要自己实现返回按钮逻辑

**建议：** 除非有特殊设计需求，否则使用原生导航栏即可！
