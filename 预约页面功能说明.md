# 景区预约页面功能说明

## 📋 页面概述

景区预约页面是一个独立的表单页面，不在底部导航栏中显示，只能通过首页的"景区预约"按钮进入。

## ✨ 功能特性

### 1. 基本信息填写
- ✅ **预约人数**：使用步进器，支持1-99人
- ✅ **联系人姓名**：必填
- ✅ **手机号码**：必填，自动验证格式
- ✅ **身份证号**：必填，支持15位和18位，自动验证校验码
- ✅ **年龄段**：选择器，可选（18岁以下、18-30岁等）
- ✅ **预约日期**：日期选择器，限制今天到3个月内

### 2. 出行方式选择
用户可以选择两种出行方式：
- 🚗 **自驾出行**
- 🚌 **旅游团**

### 3. 自驾信息（条件显示）
当选择"自驾出行"时，显示以下字段：
- ✅ **车辆类型**：轿车/SUV/商务车/客车
- ✅ **车牌号**：必填，自动验证车牌格式（支持普通车牌和新能源车牌）
- ⭕ **车辆颜色**：选填

### 4. 旅游团信息（条件显示）
当选择"旅游团"时，显示以下字段：
- ✅ **旅行社名称**：必填
- ✅ **团队编号**：必填
- ⭕ **导游姓名**：选填
- ⭕ **导游电话**：选填
- ✅ **大巴车牌号**：必填，自动验证车牌格式

### 5. 备注信息
- ⭕ **备注**：选填，最多200字，实时显示字数统计

## 🔒 表单验证规则

### 必填字段验证
1. **预约人数**：必须大于0
2. **联系人姓名**：不能为空
3. **手机号码**：
   - 不能为空
   - 必须是11位数字
   - 必须以1开头，第二位是3-9
4. **身份证号**：
   - 不能为空
   - 支持15位或18位
   - 18位身份证会验证校验码
5. **预约日期**：必须选择

### 条件必填验证

**自驾模式下：**
- 车辆类型：必选
- 车牌号：必填且格式正确

**旅游团模式下：**
- 旅行社名称：必填
- 团队编号：必填
- 大巴车牌号：必填且格式正确

## 📱 交互流程

```
首页
  ↓ 点击"景区预约"
预约表单页面
  ↓ 填写基本信息
  ↓ 选择出行方式
  ├─→ 自驾：填写车辆信息
  └─→ 旅游团：填写团队信息
  ↓ 填写备注（可选）
  ↓ 点击"提交预约"
  ↓ 表单验证
  ├─→ 验证失败：显示错误提示
  └─→ 验证成功：提交数据
      ↓ 显示"预约成功"
      ↓ 跳转到"预约记录"页面
```

## 🎨 UI特点

1. **分区设计**：使用图标+标题的分区设计，清晰明了
2. **必填标识**：必填字段前有红色星号标记
3. **条件显示**：根据出行方式动态显示/隐藏相关表单
4. **即时反馈**：
   - 输入验证失败立即提示
   - 字数统计实时更新
5. **底部固定提交**：提交按钮固定在底部，方便操作
6. **渐变按钮**：使用渐变色按钮，视觉效果更佳

## 🛠 技术实现

### 表单数据结构
```javascript
formData: {
    personCount: 1,           // 人数
    contactName: '',          // 联系人
    phone: '',                // 手机号
    idCard: '',               // 身份证号
    ageRange: '',             // 年龄段
    bookingDate: '',          // 预约日期
    travelType: 'self-driving', // 出行方式
    vehicleInfo: {            // 车辆信息
        type: '',             // 车辆类型
        plateNumber: '',      // 车牌号
        color: ''             // 车辆颜色
    },
    tourInfo: {               // 旅游团信息
        agencyName: '',       // 旅行社名称
        groupNumber: '',      // 团队编号
        guideName: '',        // 导游姓名
        guidePhone: '',       // 导游电话
        busPlateNumber: ''    // 大巴车牌号
    },
    remark: ''                // 备注
}
```

### 验证工具
已创建 `utils/validator.js` 工具类，包含：
- `validatePhone()` - 手机号验证
- `validateIdCard()` - 身份证号验证（含校验码）
- `validatePlateNumber()` - 车牌号验证
- `validateEmail()` - 邮箱验证
- `validateChineseName()` - 中文姓名验证
- `validateRange()` - 数字范围验证

## ⚠️ 注意事项
