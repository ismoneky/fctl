🚗 自驾出行

## 🔄 功能逻辑

### 1. 景区摆渡车
**特点**：
- 不需要填写车辆信息
- 不需要填写团队信息
- 适合单独游客或小团体

**流程**：
```
选择"景区摆渡车"
  ↓
继续填写其他信息（备注等）
  ↓
提交预约
```

### 2. 自驾出行
**特点**：
- 需要填写车辆信息
- 包含车辆类型、车牌号、车辆颜色

**流程**：
```
选择"自驾出行"
  ↓
显示"车辆信息"表单区块
  ↓
必填：车辆类型、车牌号
选填：车辆颜色
  ↓
提交预约
```

**车辆信息表单**：
```
┌─────────────────────────────────┐
│ 🚙 车辆信息                     │
├─────────────────────────────────┤
│ * 车辆类型: [请选择车辆类型]    │
│ * 车牌号:   [京A12345]          │
│   车辆颜色: [白色]              │
└─────────────────────────────────┘
```

### 3. 观光团
**特点**：
- 需要填写观光团信息
- 包含旅行社、团队编号、导游信息、大巴车牌号

**流程**：
```
选择"观光团"
  ↓
显示"观光团信息"表单区块
  ↓
必填：旅行社名称、团队编号、大巴车牌号
选填：导游姓名、导游电话
  ↓
提交预约
```

**观光团信息表单**：
```
┌─────────────────────────────────┐
│ 👥 观光团信息                   │
├─────────────────────────────────┤
│ * 旅行社名称: [XX旅行社]        │
│ * 团队编号:   [TG20260315001]   │
│   导游姓名:   [张三]            │
│   导游电话:   [13800138000]     │
│ * 大巴车牌号: [京A88888]        │
└─────────────────────────────────┘
```

## ✅ 表单验证

### 必填验证
```javascript
// 1. 出行方式必选
if (!this.formData.travelType) {
    提示: "请选择出行方式"
}

// 2. 自驾出行的验证
if (travelType === 'self-driving') {
    if (!vehicleInfo.type) {
        提示: "请选择车辆类型"
    }
    if (!vehicleInfo.plateNumber) {
        提示: "请输入车牌号"
    }
    if (!validatePlateNumber(plateNumber)) {
        提示: "请输入正确的车牌号"
    }
}

// 3. 观光团的验证
if (travelType === 'tour-group') {
    if (!tourInfo.agencyName) {
        提示: "请输入旅行社名称"
    }
    if (!tourInfo.groupNumber) {
        提示: "请输入团队编号"
    }
    if (!tourInfo.busPlateNumber) {
        提示: "请输入大巴车牌号"
    }
    if (!validatePlateNumber(busPlateNumber)) {
        提示: "请输入正确的大巴车牌号"
    }
}
```

## 📊 数据结构

### formData.travelType 值
```javascript
{
    'shuttle-bus': '景区摆渡车',
    'self-driving': '自驾出行',
    'tour-group': '观光团'
}
```

### 完整数据结构
```javascript
formData: {
    // ... 基本信息
    travelType: 'shuttle-bus',  // 出行方式

    // 车辆信息（仅自驾时填写）
    vehicleInfo: {
        type: 'car',           // 车辆类型
        plateNumber: '京A12345', // 车牌号
        color: '白色'           // 车辆颜色（选填）
    },

    // 观光团信息（仅观光团时填写）
    tourInfo: {
        agencyName: 'XX旅行社',        // 旅行社名称
        groupNumber: 'TG20260315001',  // 团队编号
        guideName: '张三',             // 导游姓名（选填）
        guidePhone: '13800138000',     // 导游电话（选填）
        busPlateNumber: '京A88888'     // 大巴车牌号
    }
}
```

## 🎨 样式特点

### 卡片式设计
- 每个选项独立成卡片
- 纵向排列，更清晰
- 大图标 + 文字，易于识别

### 选中效果
- 蓝色边框（#667eea）
- 浅蓝色背景（#e8f4ff）
- 蓝色文字
- 右侧勾选图标（✓）
- 阴影效果

### 未选中效果
- 灰色背景（#f8f8f8）
- 透明边框
- 黑色文字
- 无勾选图标

## 💡 优势对比

### 原设计的问题
1. 只有2个选项，不够全面
2. 横向排列在小屏幕上显示不佳
3. 单选框不够直观
4. 缺少图标，识别性差

### 新设计的优势
1. ✅ 3个选项，覆盖更多场景
2. ✅ 纵向排列，适配各种屏幕
3. ✅ 卡片式设计，更直观
4. ✅ 大图标，易于识别
5. ✅ 选中效果明显
6. ✅ 与时间段选择风格统一
