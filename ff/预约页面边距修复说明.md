## 🎯 修复原理

### 1. Box-Sizing 盒模型
```
box-sizing: border-box;
```
- 确保 padding 和 border 包含在元素的总宽度内
- 防止因为 padding 导致元素超出容器

### 2. Flex 布局优化
```
flex: 1;
min-width: 0;
```
- `flex: 1` 让元素平分空间
- `min-width: 0` 允许 flex 元素收缩到小于内容宽度
- 防止文字过长导致容器被撑大

### 3. 文字溢出处理
```
white-space: nowrap;
overflow: hidden;
text-overflow: ellipsis;
```
- 单行显示
- 隐藏溢出部分
- 显示省略号

### 4. 安全区域适配
```
padding-bottom: calc(20rpx + env(safe-area-inset-bottom));
```
- 适配 iPhone X 等带刘海屏的设备
- 确保按钮不被底部安全区域遮挡
